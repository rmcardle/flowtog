# Flowtog

*Flowtog* is a workflow tool for photographers

## Organization

* _Collection_: A folder containing subfolders and files comprising a workflow.  
* _Group_: A set of related files such as a JPEG photo, its RAW, and any edited versions.  

## Collection Directories

* _Unsorted_: Photos that have been imported but not yet been sorted or are in the process of being sorted. (JPEG+RAW)  
* _Rejected_: Photos that have been rejected. (JPEG+RAW)  
* _Photos_: Photos that have been selected. For photos that have edits, the final edited version goes here. (JPEG)  
* _Raw_: Photos that have been selected. (RAW)  
* _Originals_: Un-edited version of photos that have edited versions. (JPEG)  
* _Previous Edits_: Previous edited versions of photos that have been edited. (JPEG)  
* _Videos_: Videos and their related files.  
* _Videos/Proxy_: Proxy versions of videos.  

## Requirements

*Flowtog* requires Windows and Python 3.14 or higher. Linux and macOS are not currently supported.
